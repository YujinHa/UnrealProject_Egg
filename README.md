# UnrealProject_Egg
2019년도 Unreal 게임 제작 프로젝트

'Egg?'는 Unreal Engine으로 개발한 게임으로 인간들이 저주에 걸려 동물이 된 세계에서 주인공인 드래곤이 저주를 풀기 위해 알과 마법책을 찾아서 모험을 떠나는 게임입니다. 
숲, 동굴, 마법의 집 스테이지로 구성되며 적들과 대치하면서 아이템을 모으는 과정에 따라서 엔딩이 달라집니다.
